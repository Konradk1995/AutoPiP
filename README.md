# Auto PiP

Automatically enters Picture-in-Picture when you switch tabs or leave Safari.

Works on YouTube, Netflix, Crunchyroll, Twitch, Disney+, Prime Video, s.to -- anything with HTML5 video.

No accounts. No tracking. No data leaves your Mac.

---

## Install

### Userscript (recommended)

Works with any userscript manager. Survives restarts, no Xcode needed, works on Safari + Chrome + Firefox + Edge.

1. Install a userscript manager if you don't have one:
   - **Safari**: [Userscripts](https://apps.apple.com/app/userscripts/id1463298887) (free, open source)
   - **Chrome/Edge**: [Tampermonkey](https://www.tampermonkey.net/)
   - **Firefox**: [Tampermonkey](https://www.tampermonkey.net/) or [Greasemonkey](https://www.greasespot.net/)
2. Click **[Install Auto PiP userscript](https://github.com/Konradk1995/AutoPiP/raw/main/autopip.user.js)**
3. Confirm the install in your userscript manager

Done. Play a video, switch tabs, PiP starts automatically.

### Safari extension (native)

Requires macOS + Xcode. Two paths depending on whether you have an Apple Developer account:

**With Apple Developer account** (extension survives restarts):

```
git clone https://github.com/Konradk1995/AutoPiP.git
cd AutoPiP
make install TEAM_ID=XXXXXXXXXX
```

Replace `XXXXXXXXXX` with your Apple Developer Team ID (find it at [developer.apple.com/account](https://developer.apple.com/account) under Membership Details). Then enable in Safari > Settings > Extensions.

**Without Apple Developer account** (requires re-enabling after every restart):

```
make install
```

Safari > Develop > Allow Unsigned Extensions (password required, resets on restart), then Safari > Settings > Extensions > Auto PiP.

### Homebrew

```
brew install --cask Konradk1995/tap/autopip
```

### One-liner

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Konradk1995/AutoPiP/main/install.sh)"
```

## How it works

Play a video. Switch tabs. The video keeps playing in a floating PiP window.

- Sets `autoPictureInPicture` on the best playing video element
- Intercepts pause calls from streaming sites during tab switch (instance-level, no global prototype patching)
- Falls back to `webkitSetPresentationMode` and `requestPictureInPicture`
- Picks the best candidate by size, play state, and audio

91 lines of JavaScript. No dependencies.

## Project structure

```
extension-src/content.js    <-- single source of truth for all PiP logic
extension-src/manifest.json <-- Safari web extension manifest
autopip.user.js             <-- generated by `make userscript` (do not edit directly)
AutoPiP/                    <-- Xcode project (native Safari extension wrapper)
Makefile                    <-- build, install, generate userscript
```

The userscript and the native extension share the same `content.js`. Edit `extension-src/content.js`, then run `make userscript` to regenerate the userscript.

---

## Makefile targets

| Command | What it does |
|---|---|
| `make userscript` | Generate `autopip.user.js` from `extension-src/content.js` |
| `make install` | Build + install native extension to ~/Applications |
| `make install TEAM_ID=XXX` | Same, but signed with your Developer Team (persists after restart) |
| `make release` | Generate userscript + build + zip for GitHub release |
| `make uninstall` | Remove from ~/Applications |
| `make clean` | Delete build artifacts + generated userscript |
